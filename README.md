# Matthew McGrath's Personal Portfolio - 2026 Redesign

## About This Project

This is my personal portfolio! Here I showcase my skills and completed works as a web developer.

In this redesigned version, I adopted some of the more modern design trends. More content was also added, including a section to showcase my daily coding activities, along with a section for services I provide. Overall, I am quite pleased with how the updated version came out.
The site was also expanded to include an additional page for my resume, to make it a little more accessible for parties who want access to it.

## Tools Used

Since I wanted to keep my portfolio simple and fast, I decided to use vanilla HTML, CSS, and a tiny bit of JavaScript. Perhaps if I see the need for it, I will use React or Next.js in a future iteration.
